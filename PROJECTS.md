**Description:**
Student Finance Wallet is a comprehensive personal finance management application tailored for students. It helps track expenses, manage budgets, plan savings, and gain insights into spending patterns. The application provides an intuitive interface for financial literacy and smart money management.

**Key Features:**
- Expense tracking with categorization and tagging
- Budget planning and monitoring with visual analytics
- Savings goals setting and progress tracking
- Financial reports and spending analytics
- Transaction history with detailed search and filtering
- Multi-currency support for international students
- Mobile-responsive dashboard for on-the-go tracking

**Tech Stack:**
- Frontend: HTML5, CSS3, JavaScript with dynamic interactivity
- Backend: PHP with secure session management
- Database: MySQL with optimized queries
- Deployment: 42web hosting platform
- Architecture: Data-driven application with real-time calculations

**Achievements:**
